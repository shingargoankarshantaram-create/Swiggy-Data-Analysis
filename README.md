📊 Swiggy Data Analysis Project

📌 Project Overview

This project analyzes Swiggy food delivery data using Excel, Python, and SQL to extract meaningful insights and visualize trends.

---

🛠️ Tools Used

- Microsoft Excel (Data Cleaning & Dashboard)
- Python (Data Analysis)
- SQL (Data Querying)

---

📂 Dataset

- Swiggy_Data.csv
- Swiggy_Data.xlsx

---

📊 Key Features

- Data Cleaning & Transformation
- Pivot Table Analysis
- Interactive Dashboard
- Sales & Order Insights
